四、常见问题

1. 为什么无法同步数据？

请确认你的设备已登录 Apple 账户，且系统已启用 iCloud。若 iCloud 状态异常，应用中的同步功能可能无法正常工作。

2. 为什么没有收到有效期提醒？

请确认你已授予通知权限，并检查系统通知设置是否允许本应用发送提醒。同时请确认对应宝物已设置有效期。

3. 为什么无法恢复购买？

请确认当前 Apple 账户与购买时使用的账户一致，并在网络环境正常时重试“恢复购买”功能。如问题持续存在，请通过支持邮箱联系。

4. 为什么 TestFlight 中看不到购买信息？

首个 App 内购买项目通常需要在 App Store Connect 中随新版本一起提交审核。在相关商品尚未完成提审或审核生效前，TestFlight 环境可能无法正常获取商品信息。

TreasureMap Technical Support

Last Updated: 2026-04-09

Thank you for using TreasureMap. If you encounter feature issues, purchase restore issues, data synchronization problems, or have any questions about using the product, you can contact technical support using the methods provided on this page.

1. Support Scope

Technical support includes, but is not limited to, the following matters:

  • App installation, launch, or runtime issues
  • Issues related to QR code scanning, generation, or recognition
  • Issues with saving, editing, or deleting treasure chest and treasure records
  • Issues with photo selection, capture, display, or processing
  • iCloud / CloudKit synchronization issues
  • Local notification reminders not triggering as expected
  • Issues with Pro purchase, restore purchase, or purchase recognition

2. Contact Information

  • Support Email: joygqz@gmail.com
  • Operator: Guoqing Zhang

If you need assistance, please contact us by email using the address above.

3. Information Recommended When Contacting Support

To help improve troubleshooting efficiency, it is recommended that you include the following information in your email whenever possible:

  • App name: TreasureMap
  • App version
  • Device model
  • iOS version
  • Time of occurrence and reproduction steps
  • Detailed description of the issue
  • If convenient, related screenshots or screen recordings

4. Frequently Asked Questions

1. Why can't my data sync?

Please confirm that your device is signed in to an Apple account and that iCloud is enabled in the system. If iCloud status is abnormal, synchronization in the App may not work properly.

2. Why didn't I receive an expiration reminder?

Please confirm that you have granted notification permission and check whether the system notification settings allow the App to send alerts. Also confirm that an expiration date has been set for the relevant treasure.

3. Why can't I restore my purchase?

Please confirm that the current Apple account is the same one used for the original purchase, and try Restore Purchases again when the network connection is normal. If the issue persists, please contact us by email.

4. Why can't I see purchase information in TestFlight?

The first in-app purchase product usually needs to be submitted for review together with a new app version in App Store Connect. Before the related product has been submitted successfully or its review result has taken effect, the TestFlight environment may not be able to retrieve product information normally.

5. Privacy Related Information

For details on how the App handles images, notifications, iCloud synchronization, and purchase-related information, please refer to the Privacy Policy page.
